## 快捷键

搜索文档Ctrl + P
搜索文档（其他窗格）Ctrl + Shift + P
转到文档表Ctrl + Alt + O
快速添加Ctrl +'
显示键盘快捷键Ctrl + Alt + Shift + H
切换侧栏Alt + S
添加文档Alt + N
添加今天的文档Alt + D
向前导航Alt + 右
向后导航Alt + 左
打开知识库切换器Ctrl + Shift + Alt + O
操作系统：快速添加笔记到目前笔记下 Ctrl + Alt + Shift +' Desktop
操作系统：专注于RemNote（从别的软件切换到RN） Ctrl + Alt + R Desktop 

## 指令

```
标签 "##"
双向链接 "{{" 或者 "\ref"+"Enter"
Emoji表情 "%%" 或者 "\emoji" + "Enter"
日期 "!!" 或者 "\date"+"Enter"
别名 "\alias"+"Enter" 或者 "alt"+"A"
按照字母排序 "\sort"+"Enter"
创建TO-DO "\todo" + "Enter"
```
