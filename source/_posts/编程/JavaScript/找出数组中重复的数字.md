## 找出数组中重复的数字

新建一个 `findRepeatNumber.js` 文件，在文件里写一个名为 `findRepeatNumber` 的函数。找出数组中重复的数字。

```js
function findRepeatNumber(nums) {
  // 补充代码
}

module.exports = findRepeatNumber;
```



## 题解

```js
function findRepeatNumber(nums) {
    const set = new Set(nums)
    return nums.find(x => !set.delete(x))
}

module.exports = findRepeatNumber
```
