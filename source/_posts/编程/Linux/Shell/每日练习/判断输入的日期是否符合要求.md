## 判断输入的日期是否符合要求

1. 判断用户输入的日期，要求符合`yyyymmdd`的格式，并输出校验结果。


```sh
#/bin/bash

# author: maojunxyz
# version: 1.0
# date: 2023-10-29

if [ $# -ne 1 ] || [ ${#1} -ne 8 ];then
  echo "Usage: bash $0 yyyymmdd"
  exit 1
fi 

mydate=$1
year=${mydate:0:4}
month=${mydate:4:2}
day=${mydate:6:2}

# 将正确和错误的输出都不打印出来
if cal "$day" "$month" "$year" >/dev/null 2>/dev/null
then
  echo "The data is OK. The date is ${year}年${month}月${day}日"
else 
  echo "The data is Error."
fi
```
