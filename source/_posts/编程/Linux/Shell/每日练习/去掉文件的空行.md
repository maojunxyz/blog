## 去掉文件的空行

写一个 bash脚本以去掉一个文本文件nowcoder.txt中的空行
示例:
假设nowcoder.txt 内容如下：

```cpp
abc

567


aaa
bbb



ccc
```

你的脚本应当输出：

```c
abc
567
aaa
bbb
ccc
```



## 题解

```sh
# 方法 1
sed -n '/^$/!p' nowcoder.txt

# 方法2
awk -n '{if ($0 != "") print $0}' nowcoder.txt

# 方法 3
awk '!/^$/ {print $0}' nowcoder.txt

# 方法 4
grep -v '^$' nowcoder.txt
```
