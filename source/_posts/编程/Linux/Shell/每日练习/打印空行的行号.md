## 打印空行的行号

编写一个shell脚本以输出一个文本文件nowcoder.txt中空行的行号（空行可能连续，从1开始输出）

示例:
假设 nowcoder.txt 内容如下：

```cpp
a
b

c

d

e


f
```

你的脚本应当输出：

```c
3
5
7
9
10
```



## 题解

```sh
# 方法 1
awk -n '/^$/ {print NR}' nowcoder.txt

# 方法 2
sed -n '/^$/=' nowcoder.txt

# 方法 3
i=1
while read line; do
	if [[ $line =~ ^$ ]]; then
		echo $i
	fi		
	i=$((i+1))
done<$1
```
