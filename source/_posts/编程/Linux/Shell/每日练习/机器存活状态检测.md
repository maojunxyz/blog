## 机器存活状态检测

给定一批服务器IP，判断机器是否存活。

```sh
#!/bin/sh
# author: maojunxyz
# version: 1.0
# date: 2023-10-24
 
# 服务器IP列表
IPS=(baidu.com qq.com)
for IP in ${IPS[@]}
do
n=`ping -c5 $IP | grep "packet" |awk -F '%' '{print $1}' |awk '{print $NF}'`
if [ -z "$n" ];then
	echo "执行失败"
	exit 
else
  n1=`echo $n|sed 's/[0-9]//g'`
  if [ -n "$n1" ];then
    echo "执行失败了"
    exit 1
  fi
fi

if [ $n -ge 20 ];then
  echo "机器${IP}宕机，丢包率是${n}%"
else
  echo "机器${IP}正常，丢包率是${n}%"
fi
done
```
