## 监控站点状态

1. 写一个脚本监控网站的可用性，访问网站放回`2xx`，`3xx`状态码则代表可正常访问，其他状态码为不可正常访问。

```sh
#!/bin/sh

# author: maojunxyz
# version: 1.0
# date: 2023-10-28

code=$(curl --connect-timeout 3 -I "$1" 2>/dev/null |grep  'HTTP' |awk '{print $2}')
if echo "$code" |grep -qE '^2[0-9][0-9]|3[0-9][0-9]';then
   echo "$1可正常访问"
else
   echo "$1不可正常访问"
fi
```
