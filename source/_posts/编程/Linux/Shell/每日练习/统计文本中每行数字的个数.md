## 统计文本中每行数字的个数

```bash
#!/bin/bash

# author: maojunxyz
# version: 1.0
# date: 2023-11-03

if [ $# -ne 1 ]; then
  echo "输入的参数个数只能是1" 
  exit 1
fi

sum=0  

while read -r line; do
  # 使用sed命令将每行中的非数字字符替换为空格，然后使用wc -L统计字符数，将结果赋给变量num。
  num=$(echo "$line" | sed 's/[^0-9]//g' | wc -L)
  sum=$((sum + num)) 
done < "$1" 

echo "sum: $sum" 
```
