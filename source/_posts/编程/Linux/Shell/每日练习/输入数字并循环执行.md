## 输入数字并循环执行

1. 提示用户输入数字，并打印出输入的数字。
2. 当用户输入不是数字时提示报错信息。
3. 当用户输入字母`q`时，退出脚本。

```sh
#!/bin/bash
# author: maojunxyz
# version: 1.0
# date: 2023-10-27

while :
do
  read -p "请输入数字，当输入q时退出!" n
  num=$(echo "$n" |sed -r 's/[0-9]//g' |  wc -c)
  if [ "$n" == "q" ]
  then
    exit 0
  fi
  if [ "$num" -ne 1 ]
  then 
     echo "输入的不是数字，请重试!"
  else
     echo "你输入的数字是：$n"
  fi
done
```
